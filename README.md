Olist Sales Dashboard — Power BI

Interactive dashboard built on the Olist Brazilian E-Commerce dataset (100K+ orders, 2016–2018).

Key metrics:Total Revenue, Average Order Value, Revenue by Product Category, Orders over Time

Tools: Power BI Desktop, DAX
Dataset: Olist (Kaggle) — orders, order items, products tables
